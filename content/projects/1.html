Arch Engine

C# - OpenGL
Source

The Arch Game Engine, developed with OpenGL in C#, features an Entity Component System, Parallax Mapping, and Point Lighting. It supports scene saving and loading with JSON, includes a level editor, and boasts a runtime compiler. The engine also offers dynamic inspector components and attribute scanning, along with model mesh loading for formats such as .fbx and .obj. Additionally, it incorporates a physics engine and a file system watcher for efficient project management, all within its 10,000 lines of code.

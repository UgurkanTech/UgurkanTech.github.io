Arch Engine

C# - OpenGL
Source

The Arch Game Engine, meticulously crafted with OpenGL in C#, stands as a game engine of note. Highlighting various specifications, it features an Entity Component System, Parallax Mapping, and Point Lighting. Additionally, it supports scene saving and loading through JSON, incorporates a level editor, and boasts a Runtime Compiler. Dynamic inspector components and attribute scanning enhance its functionality, along with model mesh loading capabilities for both .fbx and .obj formats. The engine incorporates a physics engine and a project management file system watcher. Impressively, the engine is composed of 10000 lines of code, reflecting its complexity and robust design.
